PAC code 跟 STAC code 是英國在更換電信公司時使用的代碼。

![PowerMax 155x](/assets/images/20251201/cover.jpeg)


## 原因

之前租屋處因為房東不同意，無法安裝光纖網路，只好使用 EE 的 4G 吃到飽方案搭配一個無線網路分享器上網。搬到新住處後，就要著手與 EE 終止我的吃到飽 4G 合約。

在搜尋了 EE 的官方網站後，發現即使已經過了合約期滿，還是沒有辦法即時終止，必須要提前三十天告知，也就是說我還要再多繳一個月的費用才有辦法解約，實在是令人不解。在深入研究之後，發現可以使用 PAC/STAC 來快速終止合約。


## PAC 跟 STAC 是什麼？

PAC 跟 STAC 是英國的電信公司免費提供給客戶，用來更換電信公司的代碼。

PAC（Porting Authorization Code）用於更換電信公司時保持現有的號碼，也就是台灣說的攜碼跳槽。 STAC（Service Termination Authorisation Code）顧名思義，終止電信公司服務且不用保有現在的號碼，這一次我就是打算使用 STAC code 去終止我與 EE 的合約。


## 如何使用

注意：PAC 跟 STAC 的流程基本上都是一樣的

### 1. 先取得 PAC/STAC code 

與你欲終止的電信公司取得該門號的 PAC/STAC code，有些電信公司可以從網站或是 App 取得，但是大部分都可以使用簡訊：

- PAC：傳簡訊「PAC」到 65075
- STAC：傳簡訊「STAC」到 75075

應該在一分鐘內就回收到電信公司的回覆簡訊，這時請把他回傳的這個 PAC/STAC code 記著，有效期間是三十天，過期就再申請即可。

![stac code](/assets/images/20251201/sms.jpeg)

請注意，如果因為你提前解約而需要違約金的話，你現有的電信公司應該會在你申請 PAC/STAC code 時告知你。

### 2. 把這個 PAC/STAC code 給你下一個電信公司

如果你剛開始使用某間電信公司服務或是剛訂購一個 SIM，通常在一開始在註冊時，他會問你說有沒有 PAC/STAC code，但也不一定要訂購新的號碼才能做這件事，你也可以用了一陣子新號碼確定沒問題後，再去找這個 PAC/STAC 申請表單（每間電信公司有不同做法），輸入你上一個步驟收到的 code 後就完成了。

### 3. 號碼轉移通常需要多久時間？

通常需要等一至兩個工作天會生效，在等待時新舊號碼都可以正常使用，等到舊的號碼突然失去信號就代表轉移成功了。


## 最後

我將 EE 的 STAC code 直接丟給我手機原本就在使用的電信公司，隔天立刻終止服務，因為 EE 的合約也是期滿後的 rolling contract，所以沒有解約金，省下一個月的費用（£36.69）。

請自行詳閱電信公司的說明，因為細節可能因為不同時間、不同電信公司而有所差異，僅此紀錄這個方法給需要的朋友。
